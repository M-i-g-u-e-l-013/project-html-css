O que é Semântica

Semântica em HTML é basicamente dar significado ao conteúdo da página, não só aparência. 🧠 💡 O que é semântica? 👉 É usar as tags HTML de forma que o código “explique” o que o conteúdo é. 🔍 Exemplo simples ❌ Sem semântica:

Menu
Conteúdo principal
Rodapé
👉 Aqui tudo é
→ não diz nada sobre o conteúdo ✅ Com semântica:
Menu
Conteúdo principal
Rodapé
👉 Agora o código tem significado claro 🧱 Principais tags semânticas 🔹
Cabeçalho da página ou seção 🔹
Menu de navegação 🔹
Conteúdo principal 🔹
Seções do conteúdo 🔹
Conteúdo independente (ex: post, notícia) 🔹
Conteúdo lateral (ex: barra lateral) 🔹
