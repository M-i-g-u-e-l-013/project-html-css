O que é HTML ?

TML, que significa HyperText Markup Language (Linguagem de Marcação de Hipertexto), é a linguagem base utilizada para estruturar e organizar o conteúdo de sites na internet, definindo elementos como textos, imagens, vídeos e links.

O que é CSS ?

CSS (Cascading Style Sheets, ou Folhas de Estilo em Cascata) é uma linguagem de estilo utilizada para definir a aparência e o layout de páginas web. Ela separa o design (cores, fontes, espaçamento) do conteúdo estrutural (HTML), permitindo criar sites responsivos e visualmente atraentes de forma eficiente e organizada.
