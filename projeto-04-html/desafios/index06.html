O que é um iframe ?

Um iframe (abreviação de inline frame) em HTML é uma tag usada para incorporar outra página da web dentro da página atual. Ou seja, ele cria uma “janela” dentro do seu site que carrega conteúdo externo.

Para entender melhor, pense no iframe como um “site dentro do site”. Quando o navegador encontra a tag iframe, ele faz uma nova requisição HTTP separada para carregar o conteúdo especificado no atributo src. Esse conteúdo é tratado como um documento completamente independente, com seu próprio DOM, CSS, JavaScript e contexto de execução. Ou seja, o que acontece dentro do iframe não interfere diretamente na página principal — pelo menos não sem mecanismos específicos.

Vídeos em iframe:

Usar vídeos em iframe é, na prática, uma das aplicações mais comuns desse elemento. Sempre que você incorpora um vídeo do YouTube, Vimeo ou outra plataforma, quase sempre está usando um iframe por trás.

Como funciona ?

Plataformas de vídeo como YouTube e Vimeo fornecem um código pronto que você só precisa colar no seu HTML. Esse código já vem configurado com um iframe apontando para o player deles.

Pontos importantes sobre iframes:

Os iframes são um recurso clássico do HTML que continuam extremamente relevantes, mesmo com a evolução das tecnologias web.

  • Isolamento de conteúdo
  • Segurança (muito importante)
  • Política de mesma origem (Same-Origin Policy)
  • Desempenho
  • Uso mais comum: vídeos e embeds

    • Em síntese, os iframes são uma ferramenta poderosa e prática para integrar conteúdos externos de forma isolada e relativamente segura. Quando usados com cuidado
