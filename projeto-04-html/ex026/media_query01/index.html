O que são Media Queries?

Media Queries são recursos do CSS usados para deixar sites responsivos. Com elas, podemos adaptar o layout para celulares, tablets e computadores.

Elas permitem alterar tamanhos, posições, cores e diversos elementos dependendo da largura da tela do usuário.
